About Pokédex Prime

This project is a full Pokédex experience built as a modern single-page app with search, saved favorites, login, profile, and rich Pokémon details.

Data is fetched live from the official PokéAPI. All user accounts, favorites, and session data are stored locally in your browser.

Features include:

  • Smooth dark UI with animated transitions
  • Search history and random Pokémon generator
  • Favorites persisted per user
  • Profile avatar selection and team builder
  • Bonus compare mode and export support

Note: This is a frontend-only demo app. Passwords are hashed in the browser for a realistic signup flow, but this is not a substitute for production-level authentication.
